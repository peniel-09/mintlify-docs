## Description

The Mustache Robot is a classic electronics project that teaches you how to build an autonomous robot capable of detecting obstacles using its "whiskers." It doesn't use any microcontrollers, just switch-based logic!

<video autoPlay muted loop playsInline className="w-full aspect-video rounded-xl" src="/videos/demo.mp4"></video>
<Info>
  This project is ideal for beginners in analog electronics who want to understand how logic works without code.
</Info>
